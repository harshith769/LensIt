## Overview

LensIt is an AI-powered research project focused on evaluating the trustworthiness of digital content using Explainable Artificial Intelligence (XAI), Natural Language Processing (NLP), and Machine Learning.

Unlike traditional fake news classifiers, LensIt aims to provide an explainable trust score by combining linguistic analysis, source credibility signals, semantic understanding, and interpretable machine learning models.

The long-term vision is to build an AI decision-support system capable of assisting users in evaluating online information with transparent and explainable reasoning.

---

## Problem Statement

The internet generates massive amounts of information every day.

Existing misinformation detection systems generally provide only binary outputs such as:

- Fake
- Real

without explaining:

- Why was this prediction made?
- Which linguistic signals influenced the model?
- How trustworthy is the source?
- How confident is the prediction?

LensIt attempts to answer these questions using explainable AI.
