# Project Overview

LensIt is an Explainable AI framework for evaluating the trustworthiness of online textual content.

The project combines:

- Machine Learning
- Natural Language Processing
- Explainable AI
- Source Credibility Analysis
- Transformer Models

to generate interpretable trust scores for digital information.

## Vision

To build a scalable AI decision-support system capable of helping users assess online information through transparent machine learning models rather than black-box predictions.
